Install

One command in Terminal — downloads, installs, and launches automatically.

curl -sL https://milv0.github.io/QuickAccess/install.sh | bash

⚠️ First launch may not resize the window (Chrome permission). Just re-open the site and it will work.

Uninstall:

curl -sL https://milv0.github.io/QuickAccess/uninstall.sh | bash
